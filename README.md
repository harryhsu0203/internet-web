# 凱銓科技商行官方網站

弱電工程、監視系統、門禁對講、網路佈線、機房整合、公共廣播等一站式整合服務。

## 專案架構

- `index.html` - 首頁
- `services.html` - 服務項目
- `projects.html` - 實績案例
- `about.html` - 關於我們
- `contact.html` - 聯絡我們
- `thanks.html` - 表單送出成功頁
- `assets/` - 靜態資源（CSS、JS、圖片）

## 本地預覽

```bash
python3 -m http.server 8000
```

瀏覽 http://localhost:8000
